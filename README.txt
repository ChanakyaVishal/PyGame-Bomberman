**To run the game run Game.py**

Games Specs
* For the game to start run Game.py
* Frame generated as the user inputs a key
* Multiple Bombs cannot be placed
* There are 3 levels in total
* By killing all the enemies in one level, the player can advance to the next level
* Colour Scheme
	-Bomberman - Blue
	-Enemies - Red
	-Breakable Brick - Bright White
* Bomb has a timer, which is displayed as the bomb is placed
    - 2 => 1 => 0(explosion)

Requirements
* Needs Python3
* pip3 install pynput
* pynput needs the xlib library
	- sudo apt-get install xlib

Note:
* If the Screen is flickering then resize your terminal to fit the games frame
